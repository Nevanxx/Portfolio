Utility Library

_.clamp(number, lower, upper)

Clamps a number within the inclusive lower and upper bounds.

_.inRange(number, start, end)

Checks if a number is within a range (start to end).

_.words(string)

Splits a string into an array of words.

_.pad(string, length)

Pads a string on both sides if it is shorter than the specified length.

_.has(object, key)

Checks if an object contains a specified key.

_.invert(object)

Inverts the keys and values of an object.

_.findKey(object, predicate)

Finds the key of the first element that satisfies the predicate function.

_.drop(array, n)

Creates a slice of the array with `n` elements dropped from the beginning.

_.dropWhile(array, predicate)

Creates a slice of the array with elements dropped until the predicate returns false.

_.chunk(array, size)

Creates an array of elements split into groups of specified size.
